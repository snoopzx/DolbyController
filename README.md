
# DolbyController

![Logo](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExcTNrMmh6cGpibDU3MDk5djhta2hpdHo2M3hqcmlyZmY2YjN2ano0OCZlcD12MV9naWZzX3NlYXJjaCZjdD1n/sKSV28GTmnbeHLlUwF/giphy.gif)

A simple tool that lets you enable or disable Dolby Atmos for any audio device using
Designed for quick toggling and perfect for Stream Deck

'-------------'![Static Badge](https://img.shields.io/badge/Made_in-KSA-white)'-------------'

## What This Tool Does

Generates two files:

Dolby_On.bat > Enables Dolby Atmos

Dolby_Off.bat > Disables Dolby Atmos

Creates VBS launchers to run them silently

Works with any audio device (Speakers / Headphones)

Automatically detects its own folder = no hard-coded paths

Compatible with official & unofficial Stream Deck

### Why DolbyController ?

Quickly switch Dolby Atmos on/off while gaming

Create audio profiles for different devices

Integrate with automation tools

Bind Atmos toggle to any macro pad or Stream Deck button

Run Atmos silently without CMD window popping up

# Quick Usage

1 | Shortcut Toggle
```
Run > DolbyTool.exe

It will ask for Your audio device name & Output type (Speakers / Headphones)

4 Files will generated automatically
But the most important are these two files :

* silent_on.vbs
* silent_off.vbs

Just double click to run it
```
2 | Stream Deck 
```
To use it with Stream Deck

Assign silent_on.vbs & silent_off.vbs to any macro key or Stream Deck button
Hint : Add open Button from System in the side bar
```
## Important Note !!

This tool requires SNO.exe to be in the same folder as the generated BAT/VBS files.
If SNO.exe is removed, renamed, or moved to another location, the Dolby Atmos actions will not work.
